Comprehensive feedback collection and analytics system for NAAC and NBA accreditation compliance in educational institutions.

== Description ==

Elite Feedback System is a powerful WordPress plugin designed specifically for educational institutions in India to collect, analyze, and report multi-stakeholder feedback for NAAC (National Assessment and Accreditation Council) and NBA (National Board of Accreditation) accreditation processes.

= Key Features =

* **Multi-Stakeholder Support** - Collect feedback from Students, Faculty, Alumni, Employers, and Parents
* **NAAC Compliance** - Pre-built templates for all 7 NAAC criteria
* **NBA Compliance** - Program Outcome (PO) and Course Outcome (CO) feedback templates
* **Advanced Form Builder** - Create custom feedback forms with drag-and-drop question builder
* **6 Question Types** - Scale/Rating, Multiple Choice, Checkboxes, Yes/No, Text, and Textarea
* **Real-time Analytics** - Statistical analysis with averages, median, standard deviation
* **Professional Reports** - Export NAAC/NBA reports in PDF, Excel, and CSV formats
* **Anonymous Feedback** - Optional anonymity for honest responses
* **Email Automation** - Send feedback requests and confirmations automatically
* **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices

= NAAC Criteria Covered =

1. Curricular Aspects
2. Teaching-Learning and Evaluation
3. Research, Innovations and Extension
4. Infrastructure and Learning Resources
5. Student Support and Progression
6. Governance, Leadership and Management
7. Institutional Values and Best Practices

= NBA Program Outcomes =

Complete support for all 12 Program Outcomes (PO1-PO12) including:
* Engineering Knowledge
* Problem Analysis
* Design/Development of Solutions
* Modern Tool Usage
* Ethics and Professional Conduct
* And more...

= Usage =

After activation, use these shortcodes to display feedback forms:

* `[efs_feedback]` - Display all active feedback forms
* `[efs_feedback_form id="123"]` - Display a specific form
* `[efs_my_responses]` - Show user's submitted feedback history

== Installation ==

= Automatic Installation =

1. Log in to your WordPress admin panel
2. Navigate to Plugins → Add New
3. Search for "Elite Feedback System"
4. Click "Install Now" and then "Activate"

= Manual Installation =

1. Download the plugin ZIP file
2. Log in to WordPress admin panel
3. Navigate to Plugins → Add New → Upload Plugin
4. Choose the downloaded ZIP file
5. Click "Install Now" and then "Activate"

= After Activation =

1. Go to Feedback System → Settings
2. Configure your institution details
3. Create forms using NAAC/NBA templates or custom forms
4. Add shortcode `[efs_feedback]` to any page
5. Start collecting feedback!

== Frequently Asked Questions ==

= Is this plugin compatible with WordPress 6.8.3? =

Yes! The plugin is fully tested and compatible with WordPress 6.8.3.

= Can I use this for both NAAC and NBA accreditation? =

Absolutely! The plugin includes pre-built templates for both NAAC (all 7 criteria) and NBA (all 12 program outcomes).

= Does it support anonymous feedback? =

Yes, you can enable anonymous feedback for any form, ensuring honest responses from stakeholders.

= Can I export the data? =

Yes! You can export responses and reports in PDF, Excel (XLSX), and CSV formats.

= Is the plugin free? =

Yes, Elite Feedback System is completely free and open-source under GPL v2 license.

= What happens to my data if I deactivate the plugin? =

Your data is preserved! Database tables are NOT deleted on deactivation to protect your valuable accreditation data.

= Can I customize the feedback questions? =

Yes! You can use pre-built templates or create completely custom forms with your own questions.

= Does it work with any WordPress theme? =

Yes, the plugin is designed to work with any WordPress theme that follows standard WordPress coding practices.
